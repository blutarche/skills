# in-progress

Experimental skills not yet graduated to their target domain. Not installed by default —
add `in-progress` to `install.conf` to opt in.

| Skill | What it does | Target |
|-------|--------------|--------|
| [`goal-crafter`](goal-crafter/SKILL.md) | Turn a rough task into a dense, context-rich goal prompt ready for `/goal` — gathers codebase/session context, routes skills, resolves ambiguity. | `meta/` |
